Great! Now we've declared 2 new empty decks of cards, one for each player.

The next step is to step through each player's cards one by one, generate a random number as a coin flip, and use that to determine which player gets the card.

The way to generate a (pseudo)random coin flip multiple times in Solidity is as follows:

```
uint nonce = 0;
for (uint i = 0; i < 10; i++) {
    // The `% 2` at the end means "divide by 2 and take the remainder", so `rand` will end up
    // equaling either `0` or `1`
    uint rand = uint(keccak256(abi.encodePacked(now, nonce))) % 2;

    if (rand == 0) {
      // do something
    } else {
      // do something else
    }

    // We need an incrementing nonce, to make sure `rand` is different each time we run the loop
    // Otherwise the result of the random would be the same in every interation
    nonce++;
}

```

 If you need a refresher on random numbers, check out <a href="https://cryptozombies.io/en/lesson/4/chapter/4" target=_blank>Lesson 4 Chapter 4</a> of CryptoZombies course 1.

> Note/reminder: This method of generating pseudorandom numbers should not be used for gambling games, as the block producers could game the system by choosing not to publish a transaction unless it resulted in them winning a large pot. But for a simple deck shuffle like we're doing, it's good enough.

## Put It to the Test

1. First, let's delete the entire contents of the inner `for` loop we implemented in **Singleton**, so we can replace it with our own logic.

2. Our first step is to generate a random `uint` named `rand` that's either 0 or 1, just like we did above. However, instead of `nonce` in the above example, we can use `player1CardCount + player2CardCount` — since we know one of these two values will increment each time we run the `for` loop.

3. Now that we have a random number generated, we can add the card to one of the player's decks.

  a. Create an `if` / `else` statement that checks if `rand` is equal to `0`. 
  
  b. If it's true, we want to add the card to Player 1's deck.  So inside the if statement, set `player1Cards[player1CardCount]` equal to `gameState.playerStates[i].cardsInDeck[j]`.

  c. Also inside the `if` statement, increment `player1CardCount` with `player1CardCount++`.

  d. Inside the `else` statement, do the opposite: add the card to **Player 2's** deck, and increment `player2CardCount`.
