Flott! Vår zombie har nå en funksjonell nedkjølingstimer.

Deretter skal vi legge til noen ekstra hjelpemetoder. Vi har opprettet en ny fil for deg som heter `zombiehelper.sol`, som importerer`zombiefeeding.sol`. Dette vil bidra til å holde koden vår organisert.

La oss gjøre det slik at zombier får spesielle evner etter å ha nådd et visst nivå. Men for å gjøre det, må vi først lære litt mer om funksjonsmodifikatorer.

## Funksjonsmodifikatorer med argumenter

Tidligere så vi på det enkle eksempelet på `onlyOwner`. Men funksjon modifikatorer kan også ta argumenter. For eksempel:
```
// mapping for å lagre brukers alder:
mapping (uint => uint) public age;

// Modifier som krever at denne brukeren er eldre enn en viss alder:
modifier olderThan(uint _age, uint _userId) {
  require (age[_userId] >= _age);
  _;
}

// Må være eldre enn 16 å kjøre bil (i USA, minst).
// Vi kan ringe 'olderThan` modifier med argumenter som:
function driveCar(uint _userId) public olderThan(16, _userId) {
  // Litt funksjon logikk
}
```

Du kan se her at `olderThan` modifier tar argumenter akkurat som en funksjon gjør. Og at `driveCar`-funksjonen sender sine argumenter til modifikatoren.

La oss prøve å lage vår egen `modifier` som bruker zombie `level`-egenskapen for å begrense tilgangen til spesielle evner.

## Test det

1. I `ZombieHelper` lager du en `modifier` som heter `overLevel`. Det vil ta 2 argumenter, `_level` (en `uint`) og `_zombieId` (også en `uint`).

2. Funksjonen bør kontrollere at `zombies[_zombieId].level` er større enn eller lik `_level`.

3. Husk å ha den siste linjen i modifikatoren, kjøre resten av funksjonen med `_;`.
