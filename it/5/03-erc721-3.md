Ottimo, tuffiamoci nell'implementazione di ERC721!

Siamo andati avanti e abbiamo copiato la shell vuota di tutte le funzioni che implementerai in questa lezione.

In questo capitolo, implementeremo i primi due metodi: `balanceOf` ed `ownerOf`.

### `balanceOf`

```
  function balanceOf(address _owner) external view returns (uint256 _balance);
```

Questa funzione prende semplicemente un `address` e restituisce quanti token possiede quell'indirizzo.

Nel nostro caso, i nostri "token" sono zombi. Ricordi dove, nella nostra DApp, abbiamo memorizzato quanti zombi possiede un proprietario?

### `ownerOf`

```
  function ownerOf(uint256 _tokenId) external view returns (address _owner);
```

Questa funzione prende un ID token (nel nostro caso, un ID Zombie) e restituisce l `address` della persona che lo possiede.

Ancora, questo è molto semplice da implementare, poiché abbiamo già un `mapping` nella nostra DApp che memorizza queste informazioni. Possiamo implementare questa funzione in una riga, solo con un'istruzione `return`.

> Nota: Ricorda, `uint256` equivale a `uint`. Fino ad ora abbiamo usato `uint` nel nostro codice, ma qui stiamo usando` uint256` perché lo abbiamo copiato/incollato dalle specifiche.
