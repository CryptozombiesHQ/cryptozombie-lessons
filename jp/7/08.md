## プレイヤーのデッキにあるカードをループ

ここでは両プレイヤーのデッキにある各カードを反復して、全てを"Slab"に変更していく例をやっていくぞ。ちなみにSlabはファンに人気のカードだ 😉

```
for (uint i = 0; i < gameState.playerStates.length; i++) {
    for (uint j = 0; j < gameState.playerStates[i].cardsInDeck.length; j++) {
        gameState.playerStates[i].cardsInDeck[j].mouldName = "Slab"
    }
}
```

見ての通り、最初の各プレイヤーリストの反復にはネスト化された`for`ループを使っていて、そのあとで各プレイヤーのカードリストを反復している。

> 注: プレイヤーのデッキにあるカードの`mouldName`を変更することで、そのカードは完全にSlabという名前のカードに交換されます。これは単なる表示名ではありません。
